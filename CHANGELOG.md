# Changelog

## 0.2.0

- Fuente única bajo `src` y plataforma Chromium completamente generada.
- Service worker producido desde un entrypoint TypeScript canónico.
- Actualizaciones híbridas de quick-fixes con validación completa y estado v2.
- Migración compatible de whitelist y filtros propios.
- Doce locales mantenidos.
- Sistema visual oscuro compartido, foco accesible y movimiento reducido.
- Eliminación de artefactos DNR, fuentes TS y bundles duplicados del runtime.
- Suite real de unit tests, contrato de build y smoke test Playwright.
- Scripts y dependencias reducidos; auditoría de dependencias sin vulnerabilidades.
